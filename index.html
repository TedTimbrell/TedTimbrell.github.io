WorldView

WorldView was a project I came up with and worked on for my Software Engineering course (ya know, the one with all the UML diagrams and design documents). The project never ended up in quite the completed state I'd wanted it sadly, but the idea I think is a noble one. WorldView was designed to query the same URL from various locations across the globe and compare the relative output to detect if a news site is censoring or catering content for a certain audience. In my opinion, an additional aspect to fake news is simply living in our own realities. WorldView was hoping to make that more difficult.

Synchrony

Winner of my Univerity Departments "Best Senior Project Award", Synchrony was a service that hosted a legal platform for synchronous Spotify playback for multiple users using Spotify's local playback API and a Django server we set up to maintain each lobby's playback state. I'm a big fan of social acitivies and synchronous media experiences and would love for this kind of thing to return to Netflix and be properly integrated into Spotify.
